# Автоматическая загрузка данных в Yandex SpeechSense с помощью Yandex Workflows

Вы можете настроить автоматическую загрузку данных в [Yandex SpeechSense](https://yandex.cloud/ru/docs/speechsense/) с помощью [Yandex Workflows](https://yandex.cloud/ru/docs/serverless-integrations/quickstart/workflows). Настройка описана в [практическом руководстве](https://yandex.cloud/ru/docs/tutorials/serverless/auto-upload), необходимые для настройки инструменты расположены в этом репозитории.
